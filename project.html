Lab Project Github

01.

Class Portal
Design and develop a class portal that will hold your work throughout the semester. Design choices are up to you, but create something that represents you as a CTD student and your interests in technology and design. Be creative and have fun with this. You will be adding on to this web site throughout the semester, so make sure there is plenty of room to grow. There are two milestones for this assignment, but this submission page is for milestone 2 only.

02.

Project 1
Create a website or web app that is enhanced by your modern JavaScript toolkit that you have built up over the first few months of the semester.

03.

Project 2
Learn a new tool (JS library, CSS library, TypeScript, some other build tool, etc.) to add to your personal web development toolkit. Document your process of learning the new tool along the way. Show off this new tool by either adding onto your project 1 (or any other existing project), or by creating a new website or web app.
