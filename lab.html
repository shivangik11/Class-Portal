01.

Javascript and the DOM Code
A web page that uses Javascript to interact with the DOM (document object model) in the following ways:
Manipulate at least one CSS property through the DOM. Manipulate at least one other DOM property that is not a CSS property. Respond to at least two different types of events using event listeners. Both your Javascript and your CSS should be in external files (not embedded in the HTML).

02.

HTML Form
A web page with a form that does the following:
Use at least one button Use at least two input tags with different types (Links to an external site.). In addition, use at least two of the following other form controls: text box, text area, check boxes, radio buttons, list box Every form element must be within a form parent tag. There must be at least one form tag in the page. The button should use the input from the other elements to manipulate the page.
